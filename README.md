> The best module ever.


## Install

```sh
$ npm install --save passbook-server
```


## Usage

```js
var passbook-server = require('passbook-server');

passbook-server('Rainbow');
```

```sh
$ npm install --global passbook-server
$ passbook-server --help
```

```sh
# creates a browser.js
$ npm run browser
```
